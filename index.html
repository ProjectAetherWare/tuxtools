Available Commands

repairall
Full system repair, update, and reboot
optimizeall
Performance optimizer + cleanup + reboot
cleanup
Fast cleanup of temp files and logs
sysinfo
Show system, CPU, and memory info
sysupdate
Safe full system update + repair
updateapps
Update APT, Snap, and Flatpak packages
clearcache
Wipe APT, system, and user caches
flushdns
Clear DNS cache
fixperms
Fix home folder permissions
speedtest
Check internet speed
findbig
Find top 10 largest files in home
rebootnow
Instantly reboot the system
shutdownnow
Instantly power off the system
temps
Show CPU/GPU temperatures
gaming-mode
Optimize system for gaming
fps-boost
Boost FPS by disabling VSync
alias-add
Add a terminal alias
log-run
Show recent command logs
update-tuxtools
Update Tuxtools
